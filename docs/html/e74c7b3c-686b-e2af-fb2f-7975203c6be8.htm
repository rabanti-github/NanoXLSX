Metadata Properties

The Metadata type exposes the following members.

Properties
 NameDescription
Application Gets or sets the application which created the workbook. Default is NanoXLSX
ApplicationVersion Gets or sets the version of the creation application. Default is the library version of NanoXLSX. Use the format xxxxx.yyyyy (e.g. 1.0 or 55.9875) in case of a custom value.
Category Gets or sets the category of the document. There are no predefined values or restrictions about the content of this field
Company Gets or sets the company owning the document. This value is for organizational purpose. Add more than one manager by using the semicolon (;) between the words
ContentStatus Gets or sets the status of the document. There are no predefined values or restrictions about the content of this field
Creator Gets or sets the creator of the workbook. Add more than one creator by using the semicolon (;) between the authors
Description Gets or sets the description of the document or comment about it
HyperlinkBase Gets or sets the hyper-link base of the document.
Keywords Gets or sets the keywords of the workbook. Separate particular keywords with semicolons (;)
Manager Gets or sets the responsible manager of the document. This value is for organizational purpose.
Subject Gets or sets the subject of the workbook
Title Gets or sets the title of the workbook
