Enumerator

No color defined.

Automatic color (determined by application).

Rgb 

RGB/ARGB color value.

Indexed 

Legacy indexed color (0-56+).

Theme 

Theme color reference (0-11: dk1, lt1, dk2, lt2, accent1-6, hlink, folHlink).

System 

System color (used in themes).
