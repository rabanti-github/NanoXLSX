NanoXLSX.Core

NanoXLSX is a small .NET library written in C#, to create and read Microsoft Excel files in the XLSX format (Microsoft Excel 2007 or newer) in an easy and native way

Description

NanoXLSX.Core contains the core components to manipulates workbooks, worksheets, cells or styles. This package does not handle reading or writing files.

Dependencies

The core library has no dependency packages, but is the base dependency for most other (plugin) packages.
